توجيه تقني: تطبيق تغيير مسار الإرفاق الأمني وقواعد الإدارة

إعداد: آيت أمقران صالح | تاريخ: 03 نوفمبر 2025

1. الهدف

ملاحظة: تحميل الملفات المعدلة folder.zip

نقل مجلد uploads إلى /var/www/uploads لزيادة الأمان، وتطبيق قواعد الإدارة الجديدة الخاصة بمتعلمي إعادة التربية (المعرفين بـ FRAIS=1500) وأبناء العمال (المعرفين بـ PAYE=1500)، وإضافة بيانات المتعلم لوصل الإرفاق.

2. المرحلة الأولى: تغييرات الخادم (SysAdmin)

1. نقل المجلد: 
sudo mv /var/www/html/preinscription/uploads /var/www/uploads
2. ضبط الصلاحيات:

تأكد من أن خادم الويب (مثل www-data) هو المالك ولديه الصلاحيات للكتابة في هذا المجلد.

sudo chown -R www-data:www-data /var/www/uploads
sudo chmod -R 775 /var/www/uploads

4. المرحلة الرابعة: تحديث الواجهة الأمامية (Views)

الملف المستهدف: application/views/preinscription/upload/upload_form_view.php

يجب تحديث الواجهة لتعرض الحالات المختلفة (قديم، إعادة تربية، جديد) وتحدد الحقول الإجبارية الصحيحة لكل حالة.

الملف المستهدف: application/views/preinscription/upload/upload_success_view.php

يجب تحديث هذه الواجهة لإضافة بيانات المتعلم (النقطة 2 من طلب الإدارة) وتطبيق "الحل (أ)" أو "الحل (ب)" لمعالجة مشكلة الروابط القديمة والجديدة.

5. المرحلة الخامسة: حل مشكلة التوافق (البيانات القديمة والجديدة)

مشكلة: قاعدة البيانات تحتوي حاليًا على روابط كاملة (http://...). النظام الجديد يحفظ أسماء ملفات فقط. هذا سيؤدي إلى فشل الروابط القديمة.

الحل (أ): التحديث عبر SQL (موصى به)

هذا الحل ينظف قاعدة البيانات مرة واحدة. قم بتنفيذه على كل قاعدة بيانات preinscriptionXX.


-- هذا السكريبت يقوم بقص الرابط الكامل والاحتفاظ باسم الملف فقط
UPDATE student_documents
SET 
    doc_payment_receipt_url = SUBSTRING_INDEX(doc_payment_receipt_url, '/', -1)
WHERE 
    doc_payment_receipt_url LIKE 'http%';

-- كرر الأمر لباقي الأعمدة
UPDATE student_documents
SET 
    doc_birth_certificate_url = SUBSTRING_INDEX(doc_birth_certificate_url, '/', -1)
WHERE 
    doc_birth_certificate_url LIKE 'http%';

الحل (ب): التحديث عبر الكود (أقل كفاءة)

إذا كان لا يمكن تعديل قاعدة البيانات، قم بجعل الواجهة upload_success_view.php ذكية للتعامل مع الحالتين:

<?php
    $url = '';
    if (filter_var($filename_or_url, FILTER_VALIDATE_URL)) {
        // 1. بيانات قديمة (رابط كامل)، استخدمه كما هو
        $url = $filename_or_url;
    } else {
        // 2. بيانات جديدة (اسم ملف)، استخدم البوابة الآمنة
        $url = base_url('index.php/upload_documents/view_file/' . htmlspecialchars($filename_or_url));
    }
?>
